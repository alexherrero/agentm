Agent M — Asset Set

Two marks · ready to drop into the repo

Primary mark — Agent M figure. Two versions: with the ring + “AGENT M” text (badge), and without (clean figure on a black square). Both produced from your final source image, just reformatted into the sizes you need.

Secondary mark — M monogram. Cleanly redrawn in flat black & white SVG to remove the metallic / chrome treatment from the source. Same design language (two concentric rings + outlined M), now pure B&W and crisp at every size.

All sizes are square PNG, sRGB, 32-bit. Transparent variants have alpha cleanly zeroed outside the design. SVG masters included for both marks.

Where to use what

  • GitHub repo avatar / Claude repo icon: agent-m/agent-m-512.png (with ring) or agent-m/agent-m-clean-512.png (no ring). The badged version reads more clearly at small sizes thanks to the white ring; the clean version is more modern.
  • Favicon: agent-m/agent-m-32.png + agent-m-16.png — the badged version has more visual anchor at 16px.
  • Apple touch icon (180×180 target): use agent-m-256.png.
  • README header / hero: m-monogram/m-monogram-2048.png for a graphic header, or agent-m-2048.png for direct identity.
  • Dark UI / docs: m-monogram/m-monogram-transparent-*.png — the outline-M is designed to sit on any dark surface.
  • Slide decks: use the .svg wrappers — they scale to any size without quality loss.
